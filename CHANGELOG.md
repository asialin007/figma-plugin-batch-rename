# Changelog

本项目的所有重要变更都会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2026-02-27

### 新增
- 批量重命名 Figma 图层功能
- 支持添加前缀和后缀
- 支持自动编号（起始值、步长、位数）
- 支持专业切图命名规范（1x、2x、3x）
- 多语言支持（中文/英文）
- 深色/浅色主题切换
- 预览重命名结果
- 版权信息显示版本号

---

## 版本号说明

- **主版本号 (MAJOR)**: 不兼容的 API 修改
- **次版本号 (MINOR)**: 向下兼容的功能性新增
- **修订号 (PATCH)**: 向下兼容的问题修正

## 如何更新版本号

1. 修改 `package.json` 中的 `version` 字段
2. 修改 `src/ui.tsx` 中版权信息的版本号（约第 1628 行）
3. 在 `CHANGELOG.md` 中添加新版本的变更记录
4. 提交代码并推送到 GitHub
